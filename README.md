#### Course Grade Evaluator
- ![Image](link-to-image)
- Initial Project Idea:
 I created a Course Grade Evaluator program using Python in JDoodle. I made an evaluator that lets students enter their own weight factors for quizzes and assessments. This helps students calculate unweighted and weighted averages and see what letter grade they would receive in the future.

Tools I Used:
I used the Python code in JDoodle to complete it. It included loops and conditional statements. It asked students to enter quiz and assessment weights. It checked if the weights were valid and added up to 100%. The program then calculated unweighted and weighted averages and displayed the letter grade.

Challenges I Faced:
A challenge I faced was getting the code to run without errors. I kept getting incorrect outputs because some conditions and inputs were wrong. I fixed this by adjusting my conditional statements and testing inputs.

Other Resources:
I used examples from class, my notes, and instructional videos.

Result:
My final program was successful. It showed the quiz and assessment weights and checked if they added up to 100%. It repeated the questions until the student entered valid inputs. Once the weights were correct, it calculated the quiz average, assessment average, unweighted average, and weighted average. It also displayed the correct letter grade.

Summary:
This project uses Python to create a Course Grade Evaluator that checks weight inputs, calculates averages, and determines letter grades.

Summary:
This project taught me how to build a database in Aporto, link data across tables, use queries to connect information, and create transcripts.
